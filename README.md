# Portfolio

## Run locally
```
npm install
npm run dev
```

## Deploy for free (Vercel — easiest)
1. Push this folder to a new GitHub repo.
2. Go to https://vercel.com, sign in with GitHub, click "New Project".
3. Select your repo. Vercel auto-detects Vite — just click Deploy.
4. You'll get a live URL like `your-name.vercel.app`.

## Deploy for free (Netlify)
1. Push this folder to a new GitHub repo.
2. Go to https://netlify.com, sign in with GitHub, click "Add new site" → "Import an existing project".
3. Select your repo. Build command: `npm run build`, publish directory: `dist`.
4. Deploy — you'll get a live URL.
